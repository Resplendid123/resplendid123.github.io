## Semantics
{: #semantics }

Semantic HTML gives the browser and assistive technology useful structure. Tags
should describe meaning before appearance.

## Forms
{: #forms }

Forms need labels, clear inputs, and predictable submission behavior. Small
details make a large difference to usability.

## Accessibility
{: #accessibility }

Accessibility starts with native elements, readable text, keyboard support, and
honest document structure.
